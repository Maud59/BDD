**BASE DE DONNES**

- Installation de MySQL Workbench
- Visual C++ (demandé par Workbench)

- EDIT > preferences > Query Editor > X Use Uppercase Keywords (pour avoir auto-completion en majuscules)

- Connexion en localhost

- Use la DB (ctrl + entrée pour exécuter le code)
- Pour exécuter une seule ligne, il suffit d'exécuter la sélection
- (Logo etoile +, appelé snippet) Liste de requêtes favorites

une énumération permet de rentrer les données que l'on souhaite dans la liste

Pour accéder à un serveur distant, il est parfois nécessaire de se connecter à partir de la console. 
Chercher le fichier mysql.exe et le lancer à partir des commandes

insérer la commande mysql -u root -p (ne jamais mettre le mot de passe directement)

insérer un " SHOW DATABASES; "

describe role;

**REQUETE**
SELECT 
    c.customer_id, 
    firstname, 
    lastname, 
    email, 
    total_ttc,
    o.created_date
FROM
    customer AS c
        INNER JOIN
    `order`AS o ON c.customer_id = o.customer_id
ORDER BY total_ttc DESC
LIMIT 1
